# Icons

The **Icon** component provides a large set of glyphs to provide context to various aspects of your application. For a list of all available icons, visit the official [Material Design Icons page](https://pictogrammers.com/library/mdi/). To use any of these icons simply use the **mdi-** prefix followed by the icon name.
