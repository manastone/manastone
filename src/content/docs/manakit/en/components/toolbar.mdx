# Toolbar

The **Toolbar** component is pivotal to any graphical user interface (GUI), as it generally is the primary source of site navigation. The toolbar component works great in conjunction with **NavigationDrawer** and **Card**.

A **Toolbar** is a flexible container that can be used in a number of ways. By default, the **Toolbar** is 64px high on desktop and 56px high on mobile. There are a number of helper components available to use with the **Toolbar**. The **ToolbarTitle** is used for displaying a title and **ToolbarItems** allow **Btn** to extend full height.

### Toolbar title

For add title in your **Toolbar**, you can use **ToolbarTitle** components

<Sample>
  <div className="d-flex flex-column">
    <Toolbar>
      <ToolbarTitle>Title</ToolbarTitle>
    </Toolbar>
  </div>
</Sample>

### Toolbar Items

For add items in your **Toolbar**, you can use **ToolbarItems** components

<Sample>
  <div className="d-flex flex-column">
    <Toolbar>
      <ToolbarTitle>Title</ToolbarTitle>
      <ToolbarItems>Item 1</ToolbarItems>
      <ToolbarItems>Item 2</ToolbarItems>
      <ToolbarItems>Item 3</ToolbarItems>
    </Toolbar>

  </div>
</Sample>
